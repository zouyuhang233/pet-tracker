# 宠物定位器 - 网关模块 (cw_wg)

> Pet Tracker Gateway -- 基于 STM32F103ZET6 + W5500 的数据网关

---

## 模块概述

网关是宠物定位系统的边缘计算节点，负责汇聚多个传感器数据，通过 W5500 以太网模块接入网络，将处理后的数据转发至云端监控平台。

## 硬件架构

```
+-------------------------------------------+
|              STM32F103ZET6 主控            |
|                                           |
|  SPI1  <---> W5500 以太网模块 (网络接入)  |
|  SPI2  <---> 扩展传感器                    |
|  USART1 <---> 调试串口 / 数据透传          |
|  GPIO  <---> W5500 复位控制 (PG6)          |
+-------------------------------------------+
         |
         v Ethernet
+----------+        +----------+
| W5500    |        | 传感器组  |
| 以太网   |        +----------+
+----------+
         |
         v TCP
+----------+
| 云服务器  |
| :8080    |
+----------+
```

## 文件结构

```
cw_wg/
+-- APP/                    # 应用层
|   +-- int_w5500.c        # W5500 初始化与 TCP 客户端
|   +-- headfile.h
+-- BSP/                    # 板级支持包
+-- Core/                   # STM32 HAL 核心
+-- Drivers/                # HAL 驱动库
+-- MDK-ARM/               # Keil 工程文件
+-- cw_wg.ioc              # STM32CubeMX 配置
```

## 网络配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| IP 地址 | 192.168.137.100 | 设备 IP |
| 网关 | 192.168.137.1 | 默认网关 |
| 子网掩码 | 255.255.255.0 | 子网掩码 |
| 服务器 | 8.134.127.141:8080 | 云平台地址 |
